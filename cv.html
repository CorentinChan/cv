Droonity (Projet Phare)

Drooners est une plateforme communautaire pour passionnés de drones qui permet de découvrir, partager et valoriser :

  • Des spots de vol via une carte interactive
  • Ses vidéos et projets dans un portfolio public
  • Echanges entre utilisateurs et création d'events pour se retrouver

Fonctionnalités supplémentaires : carte interactive avec API Leaflet, gestion des zones de restrictions via GeoJSON, portfolios photo/vidéo, événements, chat et threads, auth JWT, modes utilisateur/admin.

React Node.js Prisma / PostgreSQL Tailwind
